# 🤖 Tars Chat App

A responsive real-time chat application built with Next.js, TypeScript, and Convex.

🔗 Live Demo: https://tars-chatapp.vercel.app

## ✨ Features
- Real-time messaging powered by Convex backend
- Built with Next.js App Router and TypeScript
- Clean, responsive UI with modern design
- Modular codebase for easy scalability

## 🛠️ Tech Stack
- **Frontend:** Next.js, TypeScript, CSS
- **Backend:** Convex (real-time database)
- **Deployment:** Vercel

## 🚀 Getting Started
npm install
npm run dev
